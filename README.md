# Croviate64
A C++ fantasy console enviroment,
it's beginner friendly and compatible with microcontrollers.
You can easily use it to build your own fantasy console. Just understand one thing that it's considered as Croviate's distro ok.

Also, it's GNU GPL so don't be selfish and let everyone have a bit of your idea.

### Here's something you need to know
| Specs      | Desktop | Embed |
|------------|---------|-------|
| Everything | Yes     | Less  |
